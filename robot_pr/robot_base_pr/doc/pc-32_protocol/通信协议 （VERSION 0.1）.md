# 通信协议 （VERSION 0.1）



## 1、数据帧定义

16位定长数据帧：

| 数据头位（0xff） | 命令位       | 数据位            | 数据尾位（0x0d） |
| ---------------- | ------------ | ----------------- | ---------------- |
| [0]（1Byte）     | [1]（1Byte） | [2]-[14] (13Byte) | [15] (1Byte)     |

数据传输方式：

 * 小端模式
