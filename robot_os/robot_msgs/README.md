# robot_msgs模块

## 1.简介

robot_msgs是RobotOS 中的一个基础功能包，提供了相关基础消息定义

## 2.msg文件

__基础数据__

* GimbalInfo.msg：云台信息
* ChassisInfo.msg：底盘信息
* RailChassisInfo：轨道底盘信息（针对哨兵）
* ShootInfo.msg：射击信息
* AttackedInfo：被攻击信息

__任务信息__

* TaskAutoAimInfo：自瞄任务信息

## 3.srv文件

* SetMode.srv：模式设置服务
